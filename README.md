# GL3

Makefile for the GL3/gl3.h; it's a Makefile script for downloading with wget and install to a system

## usage

    make
    make install

donwload "gl3.h" from "http://www.opengl.org/registry/api/gl3.h" and install to "/usr/include/GL3/gl3.h".
